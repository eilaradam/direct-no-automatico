# Direct no Automático

Página de venda do curso, em direct.laradam.com

- `index.html` — a página inteira (chat + seções). Pra editar a conversa, mexa no objeto `ROTEIRO`. Pra trocar o link de compra, na constante `CHECKOUT` no começo do script.
- `img/` — a foto da Lara e os prints do sistema.
- Números: gravados na tabela `direct_events` do Supabase do creators. Painel em creators.laradam.com/admin, aba **Direct**.
